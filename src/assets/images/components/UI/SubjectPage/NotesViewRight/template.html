But we are not the only creators of Control Systems; these systems also exist in nature, within our bodies there are numerous control systems, such as our heart which pumps the blood, our pancreas which regulates our blood sugar.




Definition of control systems



“A Control System consists of subsystems and processors (or plants), assembled for the purpose of controlling the output processes.”



1. An air conditioner maintaining the temperature of a room. 2. A car moving from rest to a speed of 100KM/h. 3. A lift moving upwards. 4. All these are Control Systems, controlling their respective outputs.






Advantages of Using Control Systems



We build the control System for four primary reasons:



1.Power Amplification


2.Remote Control


3.Convenience of Input Form


4.Compensation for Disturbance




Examples

There is a radar antenna and we need to rotate this antena thorugh a certain angle.



In this case we will use a rotating knob (present in the control room), we will rotate the knob and the antenna will rotate through the same angle.



Power Amplification


The power required to rotate is quite less than the power required to rotate the heavy antenna. In this situation, we need to amplify the power and we do this by using motors. The rotation in knob is the input to the motors which amplify the power and rotate the antenna through the same angle.



Power Amplification

The rotating knob that is used to rotate the antenna works like a remote-control element. In this way Control Systems are useful in remote locations.



Regular Control

The rotating knob that is used to rotate the antenna works like a remote-control element. In this way Control Systems are useful in remote locations.



Convenience of input form

When we rotate the knob, in order to rotate the antenna, we actually reposition the knob. We change the form of the input. We can have one more example, if we want to decrease/lower the temperature of an air-conditioned room, we push a button of its remote control.


In this way, Control System can also be used to provide convenience by changing the form of the input.



Compensation for disturbance

This is one of the most important reason to build a Control System.


Let’s understand this with the same example, already we are having a radar antenna pointing on a commanded position.


Let’s consider a case that the wind forces this antenna from its commanded position to a different position or if noise enters internally. Then, the radar antenna must be able to detect the amount of disturbance occurred and correct it by itself i.e., again it should come back to the commanded position when the effect of wind is over. We don’t change the input according to the disturbance. The Control System must be able to compensate for the disturbance by itself.
